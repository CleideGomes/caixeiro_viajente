# DESAFIO TESTE UNITÁRIOS

 O Problema do Caixeiro Viajante é um problema que tenta determinar a menor rota para percorrer uma série de cidades, retornando à cidade de origem.
 
Existem diversas formas de resolver esse problema. Uma delas é utilizando algoritmos genéticos.

Implemente os testes unitários do projeto anexado. Este projeto teve como referência o trabalho disponível em https://github.com/douglasangeli/CacheiroViajante
